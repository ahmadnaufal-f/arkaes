# Arkaes · Digital Touchpoints Reference

## arkaes.dev — Primary Portfolio

**Visual system:** Full Warm Luxe — blush + sage palette, all tokens applied.

**Fonts:** Fraunces display, Plus Jakarta Sans body, DM Mono labels.

**Motion:**
- Staggered scroll reveals (IntersectionObserver, threshold 0.08)
- Spring physics on interactive elements
- Mouse parallax on hero composition (±8–16px range)
- Hover: card scale 1.04, translateY(-2px), shadow-xs
- Cursor: custom blush dot → sage ring on hover

**Light mode only** for arkaes.dev — no dark mode toggle on the portfolio.

**Grain texture:** Fixed, 2.2% opacity, always present.

**Hero structure:**
1. Status badge (sage pulse dot + "Open to senior frontend roles")
2. Name — full name as H1, Fraunces 200
3. Role — DM Mono uppercase, blush-deep
4. Company line — Plus Jakarta Sans italic, ink-ghost
5. Blush divider rule (40px × 1px)
6. Bio paragraph
7. Actions (primary + ghost CTA)
8. Arkaes byline (bottom-left, DM Mono, parchment-d color)

**Work section structure:**
- Enterprise projects → full-width list items (confidential badge)
- Side projects / experiments → labeled "Modern Experiments", 3-col grid

**About section:**
- Arkaes mentioned naturally: *"This site is built on Arkaes — my personal
  design system and visual language for frontend projects."*
- Portrait placeholder uses AN monogram in Fraunces italic
- Sage rule at portrait bottom (counts toward sage limit)
